## Animation Patterns

### Fade In Up (On Scroll)

```
BEFORE SCROLL INTO VIEW
┌─────────────────┐
│                 │  ← Element invisible
│                 │     opacity: 0
│     (hidden)    │     translateY(30px)
│                 │
└─────────────────┘

AFTER SCROLL INTO VIEW
┌─────────────────┐
│                 │  ← Element visible
│   ┌─────────┐   │     opacity: 1
│   │ Content │   │     translateY(0)
│   └─────────┘   │     Transition: 600ms
└─────────────────┘
```

### Card Hover Effect

```
DEFAULT                    HOVER
┌─────────────┐           ┌─────────────┐
│   Card      │           │   Card      │ ↑ Lift 8px
│   Content   │     →     │   Content   │   Shadow grows
│   [Link]    │           │   [Link →]  │   Link arrow moves
└─────────────┘           └─────────────┘
 Base position              Gradient tint
                            appears
```
